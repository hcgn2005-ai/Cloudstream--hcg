# hcgn2005-ai CloudStream Repository

This repository contains the VidBox plugin for CloudStream.

## Features
- Stream movies and TV shows from VidBox
- Search functionality
- Main page with featured content

## Installation
1. Add this repository URL in CloudStream:
   `https://raw.githubusercontent.com/hcgn2005-ai/Cloudstream--hcg/main/repository.json`

2. Install the VidBox plugin
